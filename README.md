# 카카오톡 일정표 자동 수집기 📅

이 프로그램은 카카오톡 PC 버전의 톡게시판 "일정" 탭에 등록된 내용들을 자동으로 읽어서(OCR 방식) **엑셀 파일(.csv)** 로 만들어주고, 이 파일을 곧바로 **구글 캘린더에 자동 등록**해 주는 도우미입니다! 🚀

## 사전 준비 (필수사항!) 💡

1. **디스플레이 배율 확인**
   - 바탕화면에서 마우스 우클릭 -> `디스플레이 설정` -> `배율 및 레이아웃`에서 **"배율"을 100%로 설정**해주세요. (확대가 되어있으면 버튼을 제대로 못 누를 수 있습니다.)

2. **카카오톡 폰트 설정 (권장)**
   - 카카오톡 `설정` -> `화면` -> `글꼴`에서 **[바탕]** 또는 **[맑은 고딕]** 폰트를 사용하시면 글자 인식률(OCR)이 훨씬 좋아집니다.

3. **Windows 한국어 언어팩 설치 (필수!)** 🛠️
   - 이 프로그램은 **Windows 내장 OCR (WinOCR)** 엔진을 사용합니다.
   - Windows `설정` → `시간 및 언어` → `언어 및 지역`에서 **한국어**가 설치되어 있는지 확인하세요.
   - 별도의 외부 OCR 엔진 설치가 필요 없습니다! 🎉

## 사용 방법 📖

### 1단계: 카카오톡 준비
1. 카카오톡 채팅방을 열고 상단의 **[일정]** 아이콘을 눌러 목록 화면(톡게시판)을 띄워줍니다.
2. 목록 중 아무 일정을 하나 눌러서 **[상세보기]** 화면에 들어가 둡니다.

![1. 톡게시판 띄우기](readme_img/1.%20톡게시판%20띄우기.png)

### 2단계: 스크립트 실행
1. `카카오톡_일정_수집기.exe` 파일을 실행합니다.
2. "대상 창"에 현재 열려있는 카카오톡 창의 이름이 뜹니다. (만약 안 뜨면 **새로고침**을 눌러주세요!)
3. 카카오톡 방을 선택하고 **[창 고정]** 버튼을 누릅니다.
   - 이때 카카오톡 창이 화면 왼쪽 위로 이동하고 크기가 자동으로 조절됩니다. **(마우스로 옮기지 마세요!)**

![2. 프로그램 실행 후 창 고정](readme_img/2.%20프로그램%20실행%20후%20창%20고정.png)

4. "저장 이름"에 원하는 파일명(예: `우리모임일정`)을 적습니다.

### 💡 상세 설정 (필요시 입력)
- **수집 개수**: 특정 개수만큼만 수집하고 싶을 때 입력합니다. (비워두면 끝까지 수집)
- **시작일/종료일**: `YYYYMMDD` 형식으로 입력합니다. (예: `20240401`)
  - **시작일**: 입력한 날짜 **이후**의 일정만 가져옵니다.
  - **종료일**: 입력한 날짜 **이전**의 일정까지만 가져오고 멈춥니다.
- **로그 검색**: 하단의 로그창에서 특정 단어(작성자 이름, 제목 등)를 검색하여 위치를 찾을 수 있습니다.

5. **[매크로 실행 시작]** 버튼을 누릅니다.
   - 마우스가 저절로 움직이면서 일정들을 수집하기 시작합니다.
   - **주의:** 마우스가 저절로 움직이는 동안에는 컴퓨터 사용을 잠시 멈춰주세요! 마우스를 건드리면 실패할 수 있습니다.
6. 완료되면 `우리모임일정_날짜_시간.csv` 파일이 엑셀에서 열 수 있는 형태로 만들어집니다. 🎉

![3. 수집 완료](readme_img/3.%20수집%20완료.png)

### (선택) 3단계: 구글 캘린더에 자동 등록하기 🗓️

만들어진 엑셀(.csv) 파일을 이용해 내 구글 캘린더에 바로 등록할 수 있습니다!

**최초 1회 필수 설정 (Google Cloud 연동)**:
구글 캘린더에 연결하기 위한 열쇠(`credentials.json`)가 필요합니다.
1. [Google Cloud Console](https://console.cloud.google.com/) 접속
2. "새 프로젝트"를 만듭니다.
3. 좌측 메뉴에서 `API 및 서비스` -> `라이브러리` 클릭
4. **Google Calendar API** 검색 후 **[사용]** 버튼 클릭
5. 좌측 메뉴 `OAuth 동의 화면` 
   - `User Type: 외부(External)` 선택 후 만들기
   - 앱 이름: 아무거나 (예: 카카오일정봇)
   - `테스트 사용자` 항목의 [+ ADD USERS]를 눌러 **내 구글 이메일**을 추가합니다. (여기에 등록된 아이디만 사용 가능!)
6. 좌측 메뉴 `사용자 인증 정보` -> 상단 `+ 사용자 인증 정보 만들기` -> **OAuth 클라이언트 ID**
   - 애플리케이션 유형: **데스크톱 앱**
7. 생성된 항목 맨 오른쪽의 **[⬇️ JSON 다운로드]** 버튼을 눌러 파일을 받습니다.
8. 다운로드 받은 파일의 이름을 `credentials.json` 으로 바꾸고, 이 `.exe` 프로그램이 있는 폴더 안에 같이 넣어주세요!

**등록 실행 방법**:
1. 프로그램에서 **[📅 구글 캘린더 등록]** 버튼을 누릅니다.
2. 만들어진 엑셀(.csv) 파일을 선택합니다.
3. **[캘린더 ID 입력]** 창이 뜹니다.
   - **내 기본 캘린더에 등록**: 그냥 `primary` 상태로 두고 [확인]을 누릅니다.
   - **그룹/공유 캘린더에 등록**: 해당 캘린더의 ID(`~~~@group.calendar.google.com` 형태)를 입력하고 [확인]을 누릅니다.
     *(캘린더 ID 확인 방법: 구글 캘린더 좌측 목록 -> 설정 및 공유 -> '캘린더 통합' 섹션의 '캘린더 ID' 복사)*

![4. 구글 캘린더 등록 시 캘린더 ID 입력 창](readme_img/4.%20구글%20캘린더%20등록시%20표기되는%20캘린더%20id%20입력%20창.png)

4. (처음 실행 시) 웹 브라우저가 열리며 구글 로그인 화면이 나옵니다. 본인 계정으로 로그인 후 "권한 허용"을 해주세요.
   - *참고: "Google에서 확인하지 않은 앱" 경고가 뜨면 `고급` -> `~~(으)로 이동`을 누르면 됩니다.*
5. 잠시 기다리면 구글 캘린더에 자동으로 모든 일정이 등록됩니다!

![5. 캘린더 확인](readme_img/5.%20완료되면%20사진과%20같이%20캘린더에서%20해당%20일정%20확인%20가능.png)

## 오류가 날 때 🆘

* **마우스 클릭이 자꾸 엉뚱한 곳을 누를 때:** 디스플레이 배율이 100%인지 확인하고, `[창 고정]` 버튼을 다시 눌러주세요.
* **글자 인식이 이상할 때:** (예: `[일협]` -> `[일리]`) 
  - `📸 화면 캡쳐 테스트` 버튼을 눌러 어떻게 인식되는지 확인해보세요.
  - 카카오톡 폰트를 **[바탕]** 체로 바꾸면 해결될 확률이 높습니다!
  - Windows 한국어 언어팩이 제대로 설치되어 있는지 확인해 보세요. (사전 준비 3번 참고)
* **구글 캘린더 등록 중 403 에러가 날 때:** OAuth 동의 화면의 "테스트 사용자"에 본인 이메일이 정확히 추가되었는지 확인하세요.
